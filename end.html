End Connection¶

Normal case¶

When the conversation is finished, the client should send a end message over the websocket:

  • type (string): end.

Exemple:

{
    "type": "end"
}

Ava will consider as error all the ws connections which are closed without end message.

Error case¶

For various reasons, ava can send an error message. In this case, you should expect that the underlying websocket connection will be closed by us after sending this error message.
